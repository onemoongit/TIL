# Basic Linux control 

## 파일 관련


| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| ls |  | | 현재 폴더에 있는 파일들 확인 | 
| ls | -a | | 폴더에 숨겨진 파일까지 확인 | 

## 이동

| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| cd |  |Path| Path 경로로 이동한다. | 
| cd  |  | ../ | 상위 폴더로 이동한다. | 
| cd | | ~ | 루트 폴더로 이동한다.| 


## 제거

| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| rm |  | File | File 을 제거한다. | 
| rm | -r | Folder | 폴더 안의 내용을 제거한다. |
| rm | -f | Foleder | 오류를 묻지 않고 실행한다. | 

##### 보통 rm -rf (folder) 이런식으로 많이 쓰인다 하지만 조심할것!
