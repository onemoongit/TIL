# Basic Linux control 

## 파일 관련


| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| ls |  | | 현재 폴더에 있는 파일들 확인 | 
| ls | -a | | 폴더에 숨겨진 파일까지 확인 | 

## 이동

| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| cd |  |Path| Path 경로로 이동한다. | 
| cd  |  | ../ | 상위 폴더로 이동한다. | 
| cd | | ~ | 루트 폴더로 이동한다.| 


## 제거

| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| rm |  | File | File 을 제거한다. | 

## 경로

| 명령어 | 옵션 | 대상 | 설명 |
| :--- | :--- | :--- | :--- |
| pwd |  |  | 루트부터 현재 폴더의 경로 확인 |
