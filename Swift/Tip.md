# Tip for Swift or Xcode

## MARK

Swift의 문법중 사용자의 편의성을 위해 나온 문법이 MARK 이다. 이는 사용자가 보기 편하게끔 헤드라인을 잡아주는 역할을 한다.

![img](./imgs/tip_1.png)

상단의 주석을 살펴보자

- // MARK : a
- // MARK : - b

위의 2개의 형식으로 MARK 가 되어있는 것을 볼 수 있다. 보이다시피 `-`가 추가되면 상단에 하나의 경계선이 생겨서 가독성을 더욱 높일 수 있다.
